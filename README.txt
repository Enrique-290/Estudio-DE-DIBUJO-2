# Bloc de Dibujo + AutoTexto (Demo)
Abre `index.html`. Activa `AutoTexto → Demo`. Escribe una palabra y al soltar el stylus verás aparecer una **caja de texto editable** justo debajo (simula reconocimiento).

### Notas
- Esta demo NO usa IA real. Muestra el **flujo** de “trazo → texto limpio”.  
- Para IA real, conecta un reconocedor (p. ej. MyScript iink, Azure Ink, Google Vision) sustituyendo la parte donde se crea la caja por una llamada a la API y colocando el texto devuelto.

### Exportar
- PNG: botón 💾.  
- PDF: botón *Exportar PDF* (abre una vista para imprimir/guardar).
